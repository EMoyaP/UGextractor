# UGextractor

UGextractor es una aplicación de escritorio (Windows) para:

- Descargar fotos y/o videos desde sitios web.
- Descargar fotos y/o videos desde tus grupos/canales de Telegram.
- Subir fotos y/o videos locales a un grupo/canal de Telegram.

## Contenido de este repositorio

- `UGextractor_portable_v1.66.zip`: versión portable lista para ejecutar.

## Requisitos para Telegram

Para usar funciones de Telegram necesitas tus credenciales de API de Telegram:

- `api_id`
- `api_hash`
- Teléfono (primer inicio de sesión)

La app solicita autenticación de forma interactiva y guarda sesión local en tu equipo.

## Uso rápido

1. Descarga `UGextractor_portable_v1.66.zip`.
2. Descomprime en una carpeta local.
3. Ejecuta `UGextractor.exe`.
4. Configura tus parámetros en cada pestaña:
   - Web
   - Telegram (descarga)
   - Subir Telegram
